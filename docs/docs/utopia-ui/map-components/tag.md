# Tags


### Tag Object


 Option         | Type              | Default      | Required   | Description 
 ---            | ---               | ---          | ---        | ---    
 `id`           | `number`          | -            |            | *id of this tag*           
 `name`           | `string`          | -            |          | *name of this tag*           
 `color`        | `string`          | -            |            | *color for this tag*
