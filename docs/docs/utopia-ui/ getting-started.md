# Getting Started

install via npm
```
 npm install utopia-ui
```

then import in your React App
```
import UtopiaMap from 'utopia-ui'
```

use the [Map UI Component](/docs/utopia-ui/map-components/map)
```
<UtopiaMap>
   height='360px'
   width='100vw'
   center={[51.3, 9.6]}
   zoom={6}
   places={places}
   events={events}
</UtopiaMap>
```
